# Introduction

VQUIZ - A trivia quiz app built by react native & open trivia database

## Installation

Clone my repo
```bash
git clone https://github.com/iamvucms/v-quiz.git
cd v-quiz
```
Use the package manager npm

```bash
yarn
cd ios && pod install
```

## Usage
```bash
react-native run-ios
```
